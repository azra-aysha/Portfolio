Selected Work

Smart Agriculture System

The Smart Agriculture System is an AI-based platform designed to help farmers make better agricultural decisions using data analysis and machine learning. The system recommends suitable crops by analyzing environmental and soil parameters such as temperature, humidity, rainfall, nitrogen, phosphorus, potassium, and pH levels.

A Random Forest machine learning algorithm is used to analyze agricultural datasets and predict the most suitable crops for specific conditions. This helps farmers select the right crop and improve productivity and efficiency in farming.

The system also includes a plant disease detection module that uses Convolutional Neural Networks (CNN) to identify plant diseases from uploaded leaf images. Early detection of plant diseases allows farmers to take timely action and prevent crop damage.

The application integrates machine learning models built using Python, while HTML and CSS are used to design the user interface for easy interaction.

Emo Bot

Emo Bot is a real-time emotion detection system that identifies human facial expressions using computer vision and machine learning techniques. The system captures live video through a webcam and analyzes facial features to classify emotions such as happy, sad, angry, surprised, and neutral.

The project is developed using Python and OpenCV for face detection and image processing. A trained machine learning model analyzes facial patterns and predicts the emotion displayed by the user.

The system processes video frames in real time, detects faces, and displays the detected emotion instantly. Emo Bot demonstrates how artificial intelligence can be used to improve human–computer interaction in applications such as interactive systems, user experience analysis, and smart assistants.
